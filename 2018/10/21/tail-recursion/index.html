尾递归

Sun Oct 21 2018

函数的最后一个动作（通常是 return）是调用其他函数的行为。 本质是类似于在函数结尾的goto调用 由于尾调用没有涉及到普通函数调用的压栈操作，因此可以实现无限制的嵌套调用

function foo()
	return bar()
end

# 用途

在最后一个函数调用后，由于是直接退出了外部函数了，所以不需要做返回值等恢复调用栈的操作，当然也就无需在调用的时候传入调用者信息了。

# 正确的尾调用

在一些语言实现中，对于尾调用的情况，不使用额外的栈
